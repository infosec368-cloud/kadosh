# Auditoría visual con Python y Nmap

Script con interfaz gráfica (Tkinter) para ejecutar escaneo básico con `nmap` sobre una IP o dominio autorizado y mostrar hallazgos defensivos.

## Uso

```bash
python3 auditoria_seguridad_gui.py
```

## Notas

- Requiere `nmap` instalado en el sistema.
- El script **no** automatiza explotación con Metasploit ni apertura de sesiones ofensivas.
- Está pensado para análisis y hardening en contextos autorizados.
